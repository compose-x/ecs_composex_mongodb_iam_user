============
Installation
============


Stable release
--------------

To install mongodb_iam_user, run this command in your terminal:

.. code-block:: console

    $ pip install ecs_composex_mongodb_iam_user

This is the preferred method to install mongodb_iam_user, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for mongodb_iam_user can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/compose-x/ecs_composex_mongodb_iam_user

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/compose-x/ecs_composex_mongodb_iam_user/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/compose-x/ecs_composex_mongodb_iam_user
.. _tarball: https://github.com/compose-x/ecs_composex_mongodb_iam_user/tarball/master
